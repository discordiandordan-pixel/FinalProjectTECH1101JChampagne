JavaScript Cheat Sheet

My personal quick-reference notes for JavaScript

1. Variables — var, let, const

Three ways to declare variables. Prefer const by default, use let when the value needs to change, and avoid var (function-scoped and hoisted in confusing ways).

const name = "Jordan";   // cannot be reassigned
let score = 0;           // can be reassigned
var old = true;          // avoid — function-scoped

2. Data Types

JavaScript has 7 primitive types plus objects.

typeof "hello"      // "string"
typeof 42           // "number"
typeof true         // "boolean"
typeof undefined    // "undefined"
typeof null         // "object" (quirk!)
typeof Symbol()     // "symbol"
typeof 9n           // "bigint"
typeof {}           // "object"
typeof []           // "object"

3. Functions

Functions can be declared, expressed, or written as arrow functions.

// Declaration
function greet(name) {
    return "Hello, " + name;
}

// Arrow function (concise)
const greet = (name) => "Hello, " + name;

// Call it
console.log(greet("Jordan")); // Hello, Jordan

4. if / else / else if

Conditional logic — run different code depending on a condition.

const temp = 15;

if (temp > 25) {
    console.log("Hot day");
} else if (temp >= 10) {
    console.log("Nice day");
} else {
    console.log("Cold day");
}

5. for Loop

Repeat code a set number of times.

for (let i = 0; i < 5; i++) {
    console.log("Count: " + i);
}
// Prints Count: 0 through Count: 4

6. while Loop

Repeat code while a condition is true. Make sure the condition eventually becomes false or you get an infinite loop.

let count = 0;
while (count < 3) {
    console.log(count);
    count++;
}
// Prints 0, 1, 2

7. Arrays

An ordered list of values. Zero-indexed.

const cats = ["Luna", "Mochi", "Pixel"];
console.log(cats[0]);       // "Luna"
cats.push("Noodle");        // add to end
cats.pop();                 // remove last
console.log(cats.length);   // 3

8. Objects

A collection of key-value pairs. Use dot notation or bracket notation to access properties.

const cat = {
    name: "Luna",
    age: 4,
    isSpayed: true
};

console.log(cat.name);       // "Luna"
console.log(cat["age"]);     // 4

9. DOM Selection

Get references to HTML elements so JavaScript can read or change them.

// By ID
const btn = document.getElementById("myBtn");

// By CSS selector (first match)
const title = document.querySelector("h1");

// All matches
const items = document.querySelectorAll(".nav-link");

10. Event Listeners

Run code when the user does something (click, keypress, input change, etc.).

const btn = document.getElementById("myBtn");

btn.addEventListener("click", function() {
    alert("Button clicked!");
});

// Arrow function version
btn.addEventListener("click", () => alert("Clicked!"));

11. Changing DOM Content & Style

Update text, HTML, or CSS of an element from JavaScript.

const heading = document.querySelector("h1");

heading.textContent = "New Title";       // change text
heading.innerHTML = "<em>Italic Title</em>"; // change HTML

heading.style.color = "red";             // inline style
heading.classList.add("highlight");      // add CSS class
heading.classList.remove("highlight");   // remove CSS class
heading.classList.toggle("active");      // toggle CSS class

12. Template Literals

Back-tick strings that allow embedded expressions — much cleaner than string concatenation.

const name = "Jordan";
const score = 95;

// Old way
console.log("Hi " + name + ", your score is " + score);

// Template literal
console.log(`Hi ${name}, your score is ${score}`);

13. Array Methods — forEach, map, filter

Higher-order methods for working with arrays without writing manual loops.

const nums = [1, 2, 3, 4, 5];

// forEach — run code for each item
nums.forEach(n => console.log(n));

// map — transform each item, return new array
const doubled = nums.map(n => n * 2); // [2,4,6,8,10]

// filter — keep items that pass the test
const evens = nums.filter(n => n % 2 === 0); // [2,4]

14. console.log & Debugging

Essential tools for inspecting values and finding bugs. Open DevTools with F12 in any browser.

console.log("Hello");           // print a value
console.warn("Watch out!");     // yellow warning
console.error("Something broke!"); // red error
console.table([{a:1},{a:2}]);   // table view

// Pause execution in DevTools
debugger;

15. Ternary Operator

A compact one-line if/else. Format: condition ? valueIfTrue : valueIfFalse

const age = 20;
const status = age >= 18 ? "adult" : "minor";
console.log(status); // "adult"

// Equivalent if/else:
// if (age >= 18) { status = "adult"; } else { status = "minor"; }

16. Null & Undefined

undefined means a variable was declared but never assigned. null is an intentional "no value".

let x;
console.log(x);          // undefined

let y = null;
console.log(y);          // null
console.log(y == null);  // true
console.log(y === null); // true (strict)
